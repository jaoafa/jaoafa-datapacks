# 貢献する - CONTRIBUTING

## 簡潔に

1. [ここ](https://github.com/jaoafa/jaoafa-datapacks/f からリポジトリをフォークする
2. Visual Studio Code でリポジトリをクローンする
3. upstream/master をもとにブランチを作成する
4. データパックを追加・編集する
5. コミットする
6. プッシュする
7. プルリクエストを送信する
8. 承認されたら本番環境に反映される
